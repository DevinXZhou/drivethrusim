HOW to READ RECORD.log 

INTRODUCTION
One record is generated when the signal to TSP50 changed


cur_id: 	current id for the node in this queue, 
		if it is negative which means it is a delay object used to delay car push from one node to the next
		The Delay object display is used for DEBUG timing, can be ignored

waits:		how much time left to go away from current window

Output Bits: 	sequence of LED signals from VEH2 to VEH7
